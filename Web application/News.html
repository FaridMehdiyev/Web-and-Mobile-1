Why you should learn Java ? (Azerbaijanian language)

Yaradılmış hər bir proqramlaşdırma dillərinin özlərinə məxsus xüsusiyyətləri var. İstifadə etdikcə başa düşülür ki hansı proqramlaşdırma dilini harda istifadə etsən daha əlverişli olar. Məsələn, niyə Java işlətməlisən, hansı səbəblərə görə məhz Java? Bu suala bir çox cavablar var hansı ki Javanın sintaksisi çox uzun və bir o qədər də çətin olsa belə, hələdə öz hökmranlığını bazarlar da qoruyur. Statiskaya əsasən, Java dilinin digər dillərə nisbətən daha çox iş fürsətlərinə malikdir və buda Java - nın hal-hazırda önəmli dərəcə də tələb edildiyinin göstəricidir (Indeed.com – aktiv olaraq 80.000 vakansiya).
  • Java proqramlaşdırma dili bir çox müxtəlif funksiyalarla vəya fəaliyyətlərlə uyğunlaşa bilir. Məsələn, yaradılan application bir çox platformalar da işləyə bilir. Java, Java Virtual Machine (JVM) tərəfindən compile edilə bilən bir təlimat dəsti olan “Java bytecode” - da tərtib edilmişdir. Buna görədə istənilən kompyuter vəya cihazlarda Java kodlarını run eləyə bilər, ancaq komputerdə JVM yüklüdürsə. Bir söznən, Java applicationları Windows, Linux, vəya Mac komputerlərində çalışa bilir.
  • Java sürətlidir. Java ilə yazdığınız proqram əgər doğrudursa, bu o deməkdir ki sizin proqramınız digər dillərə(Python, Ruby, PHP və s.) nisbətən 10 dəfə çox sürətlidir, C++ - a nisbətəndə yavaşdır (3x - dəfə). Ancaq bəzi məqamlar var ki, ora da Java dili C++ - dan sürətli işləyir :).
  • Proqramlaşdırma da vacib olan məqamlardan biridə “secure – yəni təhlükəsiz” olmasıdır. Byte kodun, compile-time checking, Exception handling, Kriptoqrafik təhlükəsizlik (java.security.package – kitabxana) və s. kimi xüsusiyyətlərinin olması, Java proqramlaşdırma dilini təhlükəsiz edir və bu səbəblərə görə də Java dilinə bank sənayesin də daha çox üstünlük verilir.

    Proqramlaşdırma da riyaziyyat önəmlidir?

    Bəzi proqramçıların ən yaralı yerlərindən biridə “Riyaziyyat“ - dır. Riyaziyyat deyildikdə ağıla gələnlərdən biridə Logic – yəni məntiqdir. Riyazi bacarıqlarınızın yüksək olması, problem-solving qabiliyyətinin də yüksək olmasına bərabərdir. Hansı ki hər ikisində də düşünmə qabiliyyətinizi işə salırsınız və sizə verilən taskları yerinə yetirə bilərsiniz.
  • Nəyə görə proqramlaşdırma da riyaziyyat mühim rol oynayır?
  • Hansı hallar da riyazi misallardan istifadə edilir?
  • "Riyaziyyatım yaxşı deyil, proqrammist ola bilərəm?”

    • Advance mövzular ilə məşğul olmaq istəyirsinizsə, məsələn AI (Artificial Intelligence ) və ya ML ( Machine Learning ) bu zaman sizə Statistics and Probability, Differential Calculus, Linear Algebra, Numerical Analysis and Optimization, Discrete Mathematics və s. kimi kursları bilməyiniz artıq şərt olacaqdır. Məsələn, siz vectorların nə olduğunu və necə işlədiklərini bilməsəniz, o zaman sizə verilən dataların əlverişli düzənləməsinin üsulunu da edə bilməyəcəksiniz. Bəs bunların hamısını mükəmməl səviyyədə bilmək lazimdır? Cavabım : - Xeyr, hamısını mükəmməl səviyyədə bilməyinizə ehtiyac yoxdur ve hamısını da yüksək səviyyədə öyrənə bilməzsiniz. Bir nüansı da qeyd etmək istəyirəm ki, əvvəllər universitetlərdə Computer Science(Engineering və ya IT) departmenti olmamışdan öncə, komputer elmi ile məşğul olan professorlar riyaziyyat departmenindən olublar. Bəs Java kimi dillər də riyaziyyat bilməyə ehtiyac var? Bu sualın cavabı isə, siz Java proqramlaşdırma dilindən nə üçün istifadə etdiyinizden asılıdır. Ancaq, sizə Discrete Mathematisc lazımdır ki logicleri daha da yaxşı anlayasınız. Calculus və Statistics kimi kursları da bilməlisiniz ki algorithmlərin daha effektiv işlətməyinizə kömək olsun. General Mathematics də lazımdır ki, problemleri həll etmək bacarığınız olsun :) Fikrimizi cəmləşdirsək, proqramlaşdırma da əjdaha olmaq istəyirsinizsə riyaziyyat önəmli faktordu.
